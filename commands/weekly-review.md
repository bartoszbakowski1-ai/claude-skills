# /weekly-review

Run a weekly review and log it.

> Configure your tools in `CONFIG.md`. `{{GOALS}}` are your ranked North-Star goals.

## Do this

1. Read the week's session logs / notes from `{{NOTES}}` — what actually happened.
2. Score the week against `{{GOALS}}`. Use concrete metrics per goal, for example:
   - **Cashflow:** outreach sent, calls booked, proposals out, revenue closed
   - **Reputation:** content shipped (long-form, newsletter, shorts)
   - **Assets / offers:** iterations, automations shipped
   - (Swap in whatever your goals actually measure.)
3. Produce the report:
   - ✅ What got done
   - ⚠️ What didn't / blockers
   - 🎯 Top 3 priorities for next week
4. Save it to `{{NOTES}}` (your planning/review location).
5. Optional: run a memory health-check — index size, orphaned notes, contradictions, stale data — and note the result in your memory changelog.

## Entry format

`Week [date] — [one-sentence summary]`

Keep it honest and short. A weekly review you'll actually re-read beats an exhaustive one you won't.
